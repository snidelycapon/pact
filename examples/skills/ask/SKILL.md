# Ask a Question

A general-purpose request for when you need input, an opinion, or an answer from a teammate. Use this when no more specific skill type fits.

## When To Use

- You have a question that needs another person's perspective
- You want to get a gut check, recommendation, or decision from someone
- The question doesn't fit a more structured skill type

## Context Bundle Fields

| Field | Required | Description |
|-------|----------|-------------|
| question | yes | The question you're asking -- be specific |
| background | no | Relevant context the recipient needs to answer well |
| options | no | If you've identified possible answers, list them here |
| urgency | no | low, normal, or high -- defaults to normal |

## Response Structure

| Field | Description |
|-------|-------------|
| answer | Direct answer to the question |
| reasoning | Why this answer, briefly |
| caveats | Anything the sender should keep in mind |
