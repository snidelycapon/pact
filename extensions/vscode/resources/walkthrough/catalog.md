## Pact Catalog

The catalog lists all **pact types** available to your team — structured templates for common work requests.

### Built-in types

PACT ships with 10 base types:

| Type | Purpose |
|------|---------|
| **ask** | Ask a question |
| **propose** | Propose an idea for feedback |
| **share** | Share information (no response needed) |
| **request** | Request specific work |
| **handoff** | Transfer ownership |
| **check-in** | Status check-in |
| **decide** | Request a decision |
| **review** | Request a review |
| **riff** | Open-ended brainstorm |
| **try** | Suggest an experiment |

### Variants

Teams can create **variants** — specialized versions of base types (e.g. `check-in--weekly`, `review--code`). Variants inherit the base type's structure and add team-specific context fields.

### Sending a request

Click any pact type in the catalog to start composing a request with that type pre-selected.
